# Magic Rental Shop - 게임 소개서

## 게임 개요

**Magic Rental Shop**은 판타지 세계를 배경으로 한 **무기 상점 경영 로그라이크 시뮬레이션** 게임입니다. 플레이어는 돈 없는 용사들을 위한 장비 대여점을 운영하며, 점점 늘어나는 월세를 감당하며 최대한 오래 생존하는 것이 목표입니다.

### 핵심 정보
- **장르**: 경영 시뮬레이션 + 로그라이크 + 수집
- **플랫폼**: 모바일
- **플레이 방식**: 세로 모드 원터치 조작
- **목표**: 최대한 오래 생존하며 Hero 수집

## 게임의 핵심 재미

### 1. **긴장감 넘치는 경영 시뮬레이션**
- 매주 증가하는 월세 (일수 × 100골드)를 감당해야 하는 생존 압박
- 한정된 자원으로 무기 구매/제작 선택의 딜레마
- 실패하면 처음부터! 로그라이크 요소

### 2. **전략적인 매칭 시스템**
- 고객의 등급/속성/레벨을 파악해 적절한 던전 선택
- 8가지 속성 상성 관계를 활용한 전략적 무기 대여
- 성공률 계산을 통한 리스크 관리

### 3. **영구적 성장 요소 - Hero 시스템**
- 극악의 확률(1~5%)로 일반 고객이 Hero로 전환
- Hero는 영구 보유! 강화된 능력으로 더 어려운 던전 도전
- "수집욕 자극 + 로그라이크의 재도전성" 결합

## 게임 플레이 흐름

### 하루의 구성 (3개 페이즈)

#### **아침 페이즈** - 준비의 시간
- **무기 상점**: 매일 갱신되는 무기 구매 (새로고침 가능)
- **대장간**: 재료로 무기 제작 (시간 소요)
- **일일 이벤트**: 랜덤 보너스/패널티 확인

#### **낮 페이즈** - 영업의 시간
- 방문한 고객들의 정보 확인 (등급, 속성, 레벨)
- 보유 무기를 적절한 고객에게 대여
- Hero 도감 확인 및 Hero 모험 파견

#### **밤 페이즈** - 정산의 시간
- 모험 결과 확인 (성공/실패, 보상 획득)
- 무기 회수 여부 확인 (실패 시 일정 확률로 손실)
- Hero 전환 확인 (성공한 고객 중 극소수)

## 게임 특징

### 깊이 있는 전투 시스템
- **5단계 등급**: Common → Uncommon → Rare → Epic → Legendary
- **8가지 속성**: 순환 상성(Fire→Water→Thunder→Earth→Air→Ice) + 특수 속성(Light/Dark)
- **성공률 계산**: 레벨차, 등급차, 속성 상성을 모두 고려

### 다양한 수집 요소
- **무기 수집**: 등급별 다양한 무기 획득 및 관리
- **재료 수집**: 던전에서 획득한 재료로 무기 제작
- **Hero 수집**: 전체 고객 도감 완성 도전

### 리스크 & 리워드
- 높은 등급 던전 = 높은 보상 BUT 높은 실패 위험
- 좋은 무기 대여 = 성공률 상승 BUT 손실 위험
- Hero 도전 = 강력한 능력 BUT 실패 시 10일 부상

## 모바일 최적화

- **간편한 원터치 조작**: 모든 상호작용을 터치로 해결
- **짧은 플레이 세션**: 하루(3페이즈) 약 5-10분 소요
- **자동 저장**: 페이즈 전환마다 자동 저장
- **세로 모드 UI**: 한 손으로 편하게 플레이

## 타겟 유저

- 방치형 게임은 지루하지만 복잡한 게임은 부담스러운 유저
- 수집 요소와 성장의 재미를 좋아하는 유저
- 전략적 선택과 리스크 관리를 즐기는 유저
- 짧은 시간에 가볍게 즐길 수 있는 게임을 찾는 모바일 유저
